Pour nous rejoindre

Pour participer à l'Opération Rogue Wave Piercer vous devez :
  • Avoir la dernière version du jeu Arma 3
  • Avoir installé le gestionnaire de mod Arma3Sync
  • Avoir téléchargé le modset par Arma3Sync


Pour installer Arma3Sync, retrouvez le tuto d'installation des GDC.

Télécharger le modset complet de l'event via ArmA3Sync
URL autoconfig armasync: ftp://serveur02.team-gie.com/.a3s/autoconfig
Event: GDC-GIE ORWP (besoin de rien d'autre !!)
